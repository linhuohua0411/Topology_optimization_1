# 摘要

区块链点对点覆盖网络的拓扑结构直接影响区块传播效率与系统鲁棒性。在 PoS 节拍驱动、跨地域时延异构与节点动态变化并存的环境中，覆盖层容易出现传播路径低效与局部连通脆弱。现有拓扑优化研究多聚焦单一目标，且在预算约束、可实施性约束与评估口径方面缺乏一致设计，导致不同方法比较时公平性与可复现性不足。

针对上述问题，本文提出一种基于网络演化动力学的全局拓扑优化策略。该策略将演化动力学与自组织动力学串联迭代，在统一流程中联合优化传播效率、连通鲁棒性与恢复能力，并引入最大度约束、连通修复预算与修复惩罚，在性能收益与改造成本之间保持一致约束。

本文基于统一协议构建跨平台实验流程。当前已闭环主文证据为 Eth-Docker（`gradient_mode=full`，BASE + LINK/NETEM 三档攻击，共 7 个场景，`n=10`）。结果显示：相对 `M-RESI/M-STAT/M-ASIM/M-FPS`，本文方法在综合鲁棒性 `R` 上均为正差分（fair 口径范围 `+0.05596~+0.19570`），在路径长度上均为负差分（`avg_path_len` 范围 `-0.70022~-0.67044`，多数比较经 Holm 校正后显著），而 `lcc_ratio` 差分较小且部分场景饱和。上述结果支持“路径效率稳定提升 + 综合鲁棒性可解释增益”的结论。Eth-Cloud 与 BTC-Docker 统计仍在补跑流程中，当前版本不做跨平台确定性量化外推。
