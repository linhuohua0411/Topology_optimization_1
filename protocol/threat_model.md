# Threat Model（五字段结构化）

> 与 `04-问题定义与威胁模型.md` 对齐；实验 JSON 可通过 `threat_model_ref` 指向本文件及版本。

## attacker objective

- （攻击者优化目标 / 破坏目标）

## attacker knowledge

- （公开信息 / 拓扑部分可见 / 节点妥协 等）

## attacker budget

- （时间、次数、可操控链路或节点比例、算力等）

## attack modes

- （与实验矩阵中 `attack_label` 一一或一对多映射）

## evaluation protocol

- （测量窗口、基线定义、成功/失败判据、与 unified full protocol 的交叉引用）

## 非目标与边界

- （本文**不**考虑的攻击者能力或场景）
